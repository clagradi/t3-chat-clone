# T3 Chat Clone - Requirements

## Frontend Dependencies
```bash
cd t3-chat-clone
npm install
```

## Backend Dependencies
```bash
cd t3-chat-api
pip install flask flask-cors flask-jwt-extended flask-sqlalchemy bcrypt openai anthropic google-generativeai
```

## Environment Setup
Create `.env` file in t3-chat-api/ with your API keys:
```
OPENAI_API_KEY=your_openai_key_here
ANTHROPIC_API_KEY=your_anthropic_key_here
GOOGLE_API_KEY=your_google_key_here
```

## Run Instructions
1. Start backend: `cd t3-chat-api && python src/main.py`
2. Start frontend: `cd t3-chat-clone && npm run dev`
3. Open http://localhost:5173
